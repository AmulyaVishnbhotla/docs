# Usage

1. Install [Docker Engine](https://docs.docker.com/engine/installation/).
1. `cd docs`
1. `make image`
1. `make container`
1. Edit `_pages/*`.
1. `make docs`
1. View `docs/*`.
1. `git add ...`
1. `git commit`
1. `git push`
1. View http://docs.cs50.net/path/to/file.html after a few seconds.
